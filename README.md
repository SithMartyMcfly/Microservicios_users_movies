This is my first microservices proyect, I use this to know about microservices and how to join two little microservices and how to configure them properly.
I have used in this microservice the following technologies in this project:

Backend
  Java 17 - primary language
  Spring Boot 3.2.x - Framework to build the API REST
  Spring Data JPA - Access and data persistence
  Hbernate - ORM
  Spring Security 6 - Security and Authentication
  JWT (JASON Web Tokens) - bassed-tokens authentication
  ModelMapper/DTO Pattern

Database
  MySQL8
  MySQL Connector/J
  Hibernate DDL Auto - Generates/Updates database tables during development

Documentation
  Swagger UI - Interactive API documentation

Tools
  Maven - Dependency managment
  Postman/ThunderClient - Manual testing
  Git + GitHub - Version control and repositories
  VSCode/InteliJ IDEA - IDE
